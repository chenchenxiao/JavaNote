1、局部内部类要访问外围类的对象时，要用final进行修饰，在jdk1.8之后就不用写了，因为默认就用final
2、内部类的作用：
    1）、实现隐藏，普通的类是不允许使用private和protected修饰的，而内部类可以，然后通过一个方法，如getxxx来取得内部类的实例，
         这样别人就无法直接访问内部类，不知道内部类的具体实现是怎样的，而且也不知道内部类的名字，可以很好的实现隐藏
    2）、可以无条件的访问外部类的元素
    3）、可以实现“多继承”，即一个类可以有多个内部类，每个内部类继承不同的类，间接的多继承
    4）、一个类继承，实现类或接口时，可能方法会同名，使用内部类可以避免这种情况发生
